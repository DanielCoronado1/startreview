¿Qué es START review?

1El proyecto START review

Start review es una pagina web hecha para que los gamers puedan consultar reseñas o dar su opinión sobre algún videojuego.

2Nuestro objetivo

El proyecto start review tiene como objetivo mostrar reseñas de videojuegos para que otros usuarios puedan decidir si vale la pena comprar el juego o puedan dar su propia opinión de un juego en especifico y puedan compartirla con otros usuarios.
